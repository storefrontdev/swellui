## Installation

1. Install the `@radix-ui/react-select`, `@radix-ui/react-label`,`@radix-ui/react-button`, and `@radix-ui/react-radio-group` component from radix-ui:

```bash
npm install @radix-ui/react-select @radix-ui/react-label @radix-ui/react-button @radix-ui/react-radio-group
```



## Usage

Cope and paste `Purchase Options` code anywhere your product object is accessible.
