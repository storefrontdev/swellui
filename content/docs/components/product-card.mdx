## Installation

1. Install the `@radix-ui/react-select` and `@radix-ui/react-radio-group` component from radix-ui:

```bash
npm install @radix-ui/react-select
```

```bash
npm install @radix-ui/react-radio-group
```

## Usage

Cope and paste `Product Options` code anywhere your product object is accessible.
