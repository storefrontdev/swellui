## Existing Project

Install Tailwind CSS. Run the following command:

```bash
npm install -D tailwindcss postcss autoprefixer
```

Follow the official guide to [install Tailwind CSS](https://tailwindcss.com/docs/installation) in your project.

### Add dependencies

Add the following dependencies:

```bash
npm install swell-js tailwindcss-animate class-variance-authority clsx tailwind-merge lucide-react
```

[swell-js](https://developers/swell.is)
[tailwindcss-animate](https://github.com)
[class-variance-authority]()
[clsx]()


### Initialize Swell

Add a file titled `swell.js` to your `/lib` folder in your root directory.

```js

  import swell from "swell-js";

  const Swell = swell.init("spout-water", "pk_WiQPovPwyGYfwAtZdLjkg43ERKKCRCgf");

  export { Swell }
```

### Configure tailwind.config.js

Here's what my `tailwind.config.js` file looks like:

```js
const { fontFamily } = require("tailwindcss/defaultTheme")

/** @type {import('tailwindcss').Config} */
module.exports = {
  darkMode: ["class", '[data-theme="dark"]'],
  content: ["app/**/*.{ts,tsx}", "components/**/*.{ts,tsx}"],
  theme: {},
  plugins: [require("tailwindcss-animate")],
}
```

### Icons

Primitive icons are from [Lucide](https://lucide.dev).  You can replace these icons with a library of your choice by updating the `@/components/ui/icons.tsx` file.

### Add a cn helper

I use a `cn` helper to make it easier to conditionally add Tailwind CSS classes. Here's how I define it in `lib/utils.ts`:

```ts
import { ClassValue, clsx } from "clsx"
import { twMerge } from "tailwind-merge"

export function cn(...inputs: ClassValue[]) {
  return twMerge(clsx(inputs))
}
```

That's it. You can now copy and paste components into `components/ui`. Then use them into your project.
